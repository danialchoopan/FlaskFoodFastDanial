# API سفارش غذا 🍽️🚀

## معرفی
این پروژه یک **API ساده برای سفارش غذا** است که با **Flask** ساخته شده.  
امکان مدیریت سفارش‌ها، رستوران‌ها و اطلاعات کاربران رو از طریق endpointهای مختلف فراهم می‌کنه.  

## ویژگی‌ها
- ساختار ساده و سبک  
- مدیریت سفارش‌ها، رستوران‌ها و اطلاعات کاربران  
- قابل اجرا و استقرار آسان (Easy to deploy)  
- کد تمیز و قابل فهم  

## هدف
یک نمونه سبک و ساده برای یادگیری و پیاده‌سازی API با Flask.
