# راهنمای اتصال اپلیکیشن اندروید به API دانیال فست فود

این راهنما برای توسعه‌دهندگان اندروید جهت یکپارچه‌سازی اپلیکیشن موبایل با بک‌آند پایتون تهیه شده است.

## مدیریت احراز هویت

پس از فراخوانی API ورود (`/user/login`)، توکن دریافتی را در `SharedPreferences` یا `EncryptedSharedPreferences` ذخیره کنید. برای تمام درخواست‌های بعدی، این توکن را در هدر ارسال نمایید:

```java
// نمونه کد OkHttp
Request request = new Request.Builder()
  .url(apiUrl)
  .addHeader("Authorization", "Bearer " + savedToken)
  .build();
```

## ارسال موقعیت جغرافیایی

هنگام انتخاب آدرس توسط کاربر در اپلیکیشن اندروید (با استفاده از نقشه‌های بومی یا Google Maps SDK)، مختصات لتیتیود و لونگیتیود را به همراه آدرس متنی به نقطه انتهایی زیر ارسال کنید:

- **Endpoint:** `POST /user/update_location`
- **Body:**
```json
{
  "latitude": 35.1234,
  "longitude": 51.5678,
  "address": "نام محله و خیابان"
}
```

## سیستم امتیازدهی

برای ثبت امتیاز، یک Dialog در اندروید طراحی کنید که عدد ۱ تا ۵ را دریافت کند و به همراه شناسه رستوران به متد زیر ارسال کند:

- **Endpoint:** `POST /add/comment`
- **Parameters:**
    - `seller_id`: شناسه رستوران
    - `rating`: عدد صحیح بین ۱ تا ۵
    - `content`: رشته متن نظر کاربر

## نکات مهم
- تمامی پاسخ‌های خطا دارای کد وضعیت (Status Code) مناسب (400, 401, 404, 500) هستند.
- در صورت دریافت خطای 401، کاربر را مجدداً به صفحه ورود هدایت کنید (Token Expired).
